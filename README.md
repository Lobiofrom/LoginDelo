Тестовое задание - приложение для логина на сервер при помощи получения кода из СМС. 
Если код неверный, выводится сообщение об ошибке и предоставляется возможность получить код повторно.

Технологии: 
- Compose,
- DI Koin,
- Retrofit,
- Flow,
- Coroutines,
- чистая архитектура,
- многомодульность.
